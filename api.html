Bagpipe

构造器，传入限流值，设置异步调用最大并发数

Examples

var bagpipe = new Bagpipe(100);
bagpipe.push(fs.readFile, ['path', 'utf-8'], function (err, data) {
  // TODO
});

Events

  • full, 当活动异步达到限制值时，后续异步调用将被暂存于队列中。当队列的长度大于限制值的2倍或100的时候时候，触发full事件。事件传递队列长度值。
函数 Bagpipe() Bagpipe
参数 limit(Number) 并发数限制值
参数 disabled(Boolean) 是否禁用并发限制，多用于测试
代码
var Bagpipe = function (limit, disabled) {
  events.EventEmitter.call(this);
  this.limit = limit;
  this.active = 0;
  this.queue = [];
  this.disabled = !!disabled;
};
util.inherits(Bagpipe, events.EventEmitter);

push

推入方法，参数。最后一个参数为回调函数

方法 Bagpipe.prototype.push() push
参数 method(Function) 异步方法
参数 args(Mix) 参数列表，最后一个参数为回调函数。
代码
Bagpipe.prototype.push = function (method) {
  var args = [].slice.call(arguments, 1);
  var callback = args[args.length - 1];
  if (typeof callback !== 'function') {
    args.push(function () {});
  }
  if (this.disabled || this.limit < 1) {
    method.apply(null, args);
    return this;
  }
  this.queue.push({
    method: method,
    args: args
  });

  this.next();

  var upper = Math.min(this.limit * 2, 100);
  if (this.queue.length > upper) {
    this.emit('full', this.queue.length);
  }

  return this;
};
